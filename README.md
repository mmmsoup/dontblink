# dontblink

Animated wallpaper functionality based on the awesome [asetroot](https://github.com/Wilnath/asetroot) by @Wilnath

## Installation
Install dependencies:
`pacman -S imagemagick imlib2` (or whatever your package manager is)

Clone the repo and setup:
```git clone https://github.com/mmmsoup/dontblink
cd dontblink
./install.sh```
